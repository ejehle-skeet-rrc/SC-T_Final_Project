# Group Assignment 7

Financial data processing system that handles account summaries, suspicious transactions, and transaction statistics.

## Setup

1. Clone the repository
2. No additional dependencies required

## Usage

```bash
# Run the application
python main.py

# Run tests
python -m unittest discover tests
```

## Components

- **Input Handler**: Reads CSV and JSON data
- **Data Processor**: Processes and analyzes financial data  
- **Output Handler**: Generates CSV reports
